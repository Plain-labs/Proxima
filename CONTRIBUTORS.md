## How to Get Listed

Every merged pull request earns a spot here. Open an issue, claim it, ship the
work — see [CONTRIBUTING.md](./CONTRIBUTING.md) for the full workflow.

---

## Ways to Contribute

- 🔵 **Soroban** — Rust contract improvements, additional tests, upgrade paths
- 🟣 **SDK** — New client methods, better error messages, React hooks package
- 🟠 **Dashboard** — UI components, Freighter wallet flows, charts
- 📖 **Docs** — Guides, tutorials, x402 integration walkthroughs
- 🟢 **Good First Issues** — Labelled issues that need no prior context

See all open issues at [github.com/IyanuOluwaJesuloba/Proxima/issues](https://github.com/IyanuOluwaJesuloba/Proxima/issues).
